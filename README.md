
# Welcome to File Based Student Club Management System
This is a simple C program that manages student club member records using file handling. Each member’s details are stored in a separate text file named after their admission number.
Features
Add a new club member
View an existing member’s details
Stores data persistently using text files
Simple menu-driven interface

Write your name on line 6, save it, and then head back to GitHub Desktop.

How It Works
When adding a member, the program creates a .txt file using the admission number.
Member details are written into the file.
When viewing, the program reads and displays the file contents.
Invalid filename characters are removed to prevent file errors.

Requirements
C compiler (GCC recommended)

Standard C libraries (stdio.h, stdlib.h, string.h)
